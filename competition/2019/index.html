QComp 2019

The 2019 Comparison of Tools for the Analysis of Quantitative Formal Models (QComp 2019) is part of the TACAS 2019 Toolympics. It is the first friendly competition among verification and analysis tools for quantitative formal models, drawing its benchmarks from the Quantitative Formal Model Repository and comparing performance, versatility and usability.

Setup

QComp 2019 is a flexible and friendly event. We want to include as many tools as possible: Every tool that is able to solve a significant subset of the benchmarks is welcome, even if it may not support all model types, the JANI format, or all included kinds of properties. For example, a tool specialising in the analysis of stochastic Petri nets will not be expected to solve JANI Markov chain models. All tools will be evaluated on a central server using parameters/command lines provided by the participants. We expect the deadline for final tool submissions – i.e. for a tool to be tested on the benchmarks with fixed parameters/commands – to be in January 2019. However, this may change subject to the exact overall deadlines of the TACAS Toolympics.
